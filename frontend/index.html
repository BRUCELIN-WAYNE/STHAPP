乳酸分级测试方案

乳酸分级测试是经典的耐力评估方法，旨在确定运动员在不同运动强度下的生理反应，特别是血乳酸浓度的拐点。这些拐点对于精确定义训练区间至关重要。

标准化测试方案 (例如, 5-7 x 200m):
  1. 准备: 泳池、秒表、校准过的便携式血乳酸分析仪。测试前24小时避免高强度训练。
  2. 热身: 进行10-15分钟的低强度有氧热身。
  3. 递增负荷: 进行5-7组200米游泳，组间休息60秒采血。强度应逐级递增直至力竭。
    (推荐) 使用下方的“个性化配速建议”工具，根据您的200米个人最好成绩生成标准的递增配速计划。
  4. 主动恢复 (可选): 在最后一组力竭后，进行10-15分钟的极低强度主动恢复。测量恢复前后的血乳酸，以计算乳酸清除率。
方案改编自: Olbrecht, J. (2013). *The Science of Winning*.

临界游泳速度 (CSS)

临界游泳速度 (CSS) 在概念上被定义为游泳者在理论上可以长时间持续运动而不会达到力竭状态的最高速度。重要的是，通过200/400米测试计算出的CSS，其配速通常显著快于真正的无氧阈 (MLSS)，因此它更准确地反映了最大代谢稳态 (Maximal Metabolic Steady State, MMSS)。

测试方法:

测试包括一次400米和一次200米的最大努力计时游，两次测试之间进行充分的恢复。

核心公式:

CSS 配速 (/100米) = (400米时间 - 200米时间) / 2

模型源自《科学化游泳训练区间设定指南》报告。
